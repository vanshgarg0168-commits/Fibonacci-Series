# Fibonacci Series

## Project Overview

This beginner-friendly Python project generates the Fibonacci Series
based on the number of terms entered by the user. It helps students
understand looping concepts, variable updates, and sequence generation.

## Objective

Practice: - Variables - User Input - Type Conversion - Arithmetic
Operators - `for` Loop - `range()` Function

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

### Libraries Used

-   No external libraries are required.

## Features

-   Accept the number of terms from the user.
-   Generate the Fibonacci Series.
-   Display the series in sequence.

## How to Run

1.  Open the notebook.
2.  Run all cells.
3.  Enter the number of terms.

## Concepts Covered

-   Variables
-   User Input
-   Type Conversion
-   Arithmetic Operators
-   `for` Loop
-   `range()` Function

## Learning Outcomes

-   Generate the Fibonacci sequence.
-   Use loops effectively.
-   Improve logical thinking.
